# 🎮 찬재수학 (Gugudan Challenge)

**찬재수학**은 아이들을 위한 인터랙티브 수학 퀴즈 웹 애플리케이션입니다. 아름다운 3D 배경과 함께 즐겁게 사칙연산을 학습할 수 있습니다.

---

## ✨ 주요 기능 (Key Features)

- **맞춤형 사용자 경험**: '찬민', '재민' 등 사용자에 따라 최적화된 연산 유형 제공
- **몰입감 넘치는 디자인**: Spline 3D 배경과 유리 질감(Glassmorphism) UI의 조화
- **실시간 피드백**: 정답/오답 시 시각적 효과 및 사운드 피드백 제공
- **데이터 기반 기록 관리**: Supabase를 통한 실시간 순위 및 기록 저장
- **반응형 대시보드**: 누적 기록 확인을 통한 학습 성취도 체감

---

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **Design**: Spline (3D Low-poly Scene)
- **Backend / Database**: Supabase (PostgreSQL)
- **Typography**: Outfit, Noto Sans KR (Google Fonts)

---

## 🚀 시작하기 (Getting Started)

1. 이 저장소를 클론합니다.
   ```bash
   git clone https://github.com/your-username/gugudan-challenge.git
   ```
2. `index.html` 파일을 브라우저에서 엽니다.
3. Supabase 연동이 필요한 경우 `script.js`의 `SUPABASE_URL`과 `SUPABASE_ANON_KEY`를 설정하세요.
